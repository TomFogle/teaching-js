Trees and Termites, by Leo and Tom

Instructions:

There are ten trees in the forest. Their names are:

  • tree1: Brutus
  • tree2: Domingo
  • tree3: Cassiopeia
  • tree4: Luna
  • tree5: Archimedes
  • tree6: Layla
  • tree7: Mark
  • tree8: Jeff
  • tree9: Geoff
  • tree10: Sasha

Every tree has mass(kg).

Every tree has leaves and termites.

Every day, termites remove some of the tree's mass.

Every day, leaves add mass to the tree via photosynthesis.

Use the directions below to see what happens to the trees' mass after one day and one year!

  • Type treeX = Learn about a specific tree! Example: tree6
  • Type leoTomForest.day(true) = See all forest data for the day!
  • Type leoTomForest.year() = See all forest data for the year!
  • Type treeX.day(true) = See a specific tree's data for the day! Example: tree6.day(true)
  • Type treeX.year() = See a specific tree's data for the year! Example: tree3.year()
